1.- Instalar nasm, con la siguiente instrucción:  
sudo apt update 
sudo apt install nasm -y 

2.- Revisar la versión de nasm instalado: 
nasm -v 

3.- Crear una carpeta cuyo nombre sea “ejemplo”: 
mkdir ejemplo

4.- Cambiar del directorio /workspaces para que la carpeta nueva sea este, con:  
cd /workspaces/OAC_2026_01/ejemplo/  

5.- Comprobar que este cambio se haya realizado con: 
pwd

6.- Crear un archivo con extensión .asm y con nombre “ejemplo”: 
touch ejemplo.asm

7.-  Ensamblar el archivo creado anteriormente: 
nasm -f elf ejemplo.asm 

8.-  Enlazar el objeto creado: 
ld -m elf_i386 -s -o ejemplo ejemplo.o 

9.-  Ejecutar el programa obtenido: 
./ejemplo
